1. Устоновите Node.js

2. Откройте фаил cmd.exe

3. Напишите npm install express body-parser

4. Запуститите Server_Start

5. Откройте сайт: http://localhost:3000


Что за что Отвечает?


В текстовый документ Password.txt будет логин и пароль который вы ввели.

Чтобы закрыть сервер просто нажмите на Server_Close

Server.js отвечает за работу сервера

Package.json и Package-lock.json системные их трогать не нужно

В папке Public хранится находится index.html

node_modules системный трогать не нужно
